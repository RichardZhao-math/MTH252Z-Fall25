    Section 5.4 The Fundamental Theorem of Calculus, Part I

    Subsection A Corollary of the Fundamental Theorem of Calculus

    Well the Fundamental Theorem of Calculus, Part II, in SectionÂ 5.5, isnâ€™t the theorem we use a lot. Instead, we use the Fundamental Theorem of Calculus, Part I, a lot to help us evaluate definite integrals. This is actually a (most useful) corollary of the Fundamental Theorem of Calculus. The theorem is presented in your textbook page 318. I also included it below:
    ðŸ”—

    Corollary 5.4.1.

    Assume that \(a < b\) and that \(f\) is continuous on \([a,b]\text{.}\) If \(F\) is an antiderivative of \(f\) on \([a,b]\text{,}\) then
    \begin{equation} \int_a^b f(x)\, dx = F(b) - F(a)\tag{5.4.1} \end{equation}
    ðŸ”—
    ðŸ”—
    Corollary, in math, means a result that follows immediately from a big theorem. So we should be able to prove this theorem quickly if we know how the Fundamental Theorem of Calculus works.
    ðŸ”—
    Let \(f\) be a continuous function. By the Fundamental Theorem of Calculus, we know that its accumulation function, \(A\text{,}\) is an antiderivative of \(f\) (since \(A'(x) = f(x)\)). Furthermore, we know that ALL antiderivatives of a function should only differ by a constant. That is, if \(F\) is an arbitrary antiderivative of \(f\text{,}\) then \(A(x) = F(x) + C\text{.}\)
    ðŸ”—
    We will prove this corollary through the accumulation function. Your textbook approaches the proof using the Mean Value Theorem, which is a different approach. Feel free to read through it if you want a different perspective of why this corollary is true.
    ðŸ”—
    Letâ€™s define the accumulation function
    \begin{equation*} A(x) = \int_c^x f(t)\, dt \end{equation*}
    The reason why we used \(c\) as the lower limit here is because \(a\) is taken for something else. In math, we donâ€™t want to label two different thing using the same letter to avoid confusion. I, again, coded a pretty picture to help you visualize the situation.
    ðŸ”—
    Using colors, we know that \(A(b)\) represents the area of the region on \([c,b]\text{,}\) which is the area of the green and orange regions combined. \(A(a)\) represents the area of the region on \([c,a]\text{,}\) which is the area of the green region.
    ðŸ”—
    We know that \(\int_a^b f(x)\, dx\) represents the area of the region on \([a,b]\text{,}\) which is the area of the orange region. A simple subtraction tells us that
    \begin{equation*} \int_a^b f(x)\, dx = A(b) - A(a) \end{equation*}
    This result is super close to the corollary already!
    ðŸ”—
    Recall \(F(x) = A(x) + C\text{.}\) Then we obtain
    \begin{align*} \int_a^b f(x)\, dx \amp= A(b) - A(a) \\ \amp= \left(F(b) + C\right) - \left(F(a) + C\right) \\ \amp= F(b) + C - F(a) - C \\ \amp= F(b) - F(a) \end{align*}
    We proved another big result in calculus!
    ðŸ”—
    The last step in our proof is another reason why we donâ€™t need a \(+C\) when evaluating definite integral. The \(+C\) will be canceled!
    ðŸ”—
    ðŸ”—

    Subsection Why is this corollary important?

    Note that the definite integrals allow us to compute the area under the curve. But finding the Riemann sum is a painful process, especially without the help of the technology.
    ðŸ”—
    Whereas we can find the indefinite integral (or the general antiderivative) of a function. But it has nothing to do with the area.
    ðŸ”—
    This corollary gives us a practical way to connect these two ideas together! We can evaluate the definite integral by (1) find the antiderivative of the integrand, and (2) plug in the limits of integration and do the subtraction. In fact, some textbooks call this result the evaluation theorem to emphasize the fact that we can evaluate definite integral using this result.
    ðŸ”—
    In notation, we have
    \begin{equation*} \int_a^b f(x)\, dx = F(x)\bigg|_a^b = F(b) - F(a)\qquad \text{where}\quad \int f(x)\, dx = F(x) + C \end{equation*}
    We will be using this idea to evaluate the definite integrals mostly in this class.
    ðŸ”—
    ðŸ”—

    Subsection Examples on Applying FTC Corollary

    We will work through many examples to evaluate definite integrals using the corollary of FTC. Make sure you are comfortable with this way of evaluating definite integrals!
    ðŸ”—
    In the following examples, I will show all the steps. Make sure you understand why each step is being done (either using properties of definite integral, FTC, purely algebra, or purely arithmetic).
    ðŸ”—

    Example 5.4.3.

    Evaluate the definite integral \(\displaystyle \int_0^2 \left(x^4 - \frac{3}{4}x^2 + \frac{2}{3}x - 1\right)\, dx \)
    ðŸ”—
    Hint.
    By the FTC, we will need to first find an antiderivative of the integrand, and then plug in the limits of integration and do the subtraction.
    ðŸ”—
    So what is an antiderivative of the integrand \(y = x^4 - \dfrac{3}{4}x^2 + \dfrac{2}{3}x - 1\text{?}\)
    ðŸ”—
    ðŸ”—
    Solution.
    \begin{align*} \int_0^2 \left(x^4 - \frac{3}{4}x^2 + \frac{2}{3}x - 1\right)\, dx\amp = \int_0^2 x^4\, dx - \frac{3}{4}\int_0^2 x^2\, dx + \frac{2}{3}\int_0^2 x\, dx - \int\, dx \\ \amp= \frac{x^5}{5}\bigg|_0^2 - \frac{3}{4}\cdot \frac{x^3}{3}\bigg|_0^2 + \frac{2}{3}\cdot \frac{x^2}{2}\bigg|_0^2 - x\bigg|_0^2 \\ \amp= \left(\frac{2^5}{5} - \frac{0^5}{5}\right) - \frac{3}{4}\left(\frac{2^3}{3} - \frac{0^3}{3}\right) + \frac{2}{3}\left(\frac{2^2}{2} - \frac{0^2}{2}\right) - (2 - 0) \\ \amp= \frac{32}{5} - 2 + \frac{4}{3} - 2 \\ \amp= \frac{56}{15} \end{align*}
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Example 5.4.4.

    Evaluate the definite integral \(\displaystyle \int_1^2 \frac{1}{x^2}\, dx \)
    ðŸ”—
    Hint.
    By the FTC, we will need to first find an antiderivative of the integrand, and then plug in the limits of integration and do the subtraction.
    ðŸ”—
    So what is an antiderivative of the integrand \(y = \dfrac{1}{x^2}\text{?}\)
    ðŸ”—
    ðŸ”—
    Solution.
    \begin{align*} \int_1^2\frac{1}{x^2}\, dx \amp= \int_1^2 x^{-2}\, dx \\ \amp= -x^{-1}\bigg|_1^2 \\ \amp= \left(-2^{-1}\right) - \left(-1^{-1}\right) \\ \amp= -\frac{1}{2} + 1 \\ \amp= \frac{1}{2} \end{align*}
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Example 5.4.5.

    Evaluate the definite integral \(\displaystyle \int_0^1 x\left(\sqrt[3]{x} + \sqrt[4]{x}\right)\, dx \)
    ðŸ”—
    Hint.
    By the FTC, we will need to first find an antiderivative of the integrand, and then plug in the limits of integration and do the subtraction.
    ðŸ”—
    So what is an antiderivative of the integrand \(y = x\left(\sqrt[3]{x} + \sqrt[4]{x}\right)\text{?}\)
    ðŸ”—
    ðŸ”—
    Solution.
    \begin{align*} \int_0^1 x\left(\sqrt[3]{x} + \sqrt[4]{x}\right)\, dx \amp= \int_0^1 x\left(x^\frac{1}{3} + x^\frac{1}{4}\right)\, dx\\ \amp= \int_0^1 \left(x^\frac{4}{3} + x^\frac{5}{4}\right)\, dx \\ \amp= \left[\frac{3}{7}x^\frac{7}{3} + \frac{4}{9}x^\frac{9}{4}\right]_0^1 \\ \amp= \left(\frac{3}{7}(1)^\frac{7}{3} + \frac{4}{9}(1)^\frac{9}{4}\right) - \left(\frac{3}{7}(0)^\frac{7}{3} + \frac{4}{9}(0)^\frac{9}{4}\right) \\ \amp= \frac{3}{7} + \frac{4}{9} \\ \amp= \frac{55}{63} \end{align*}
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Example 5.4.6.

    Evaluate the definite integral \(\displaystyle \int_{-1}^1 t\left(1 - t\right)^2\, dt \)
    ðŸ”—
    Hint.
    By the FTC, we will need to first find an antiderivative of the integrand, and then plug in the limits of integration and do the subtraction.
    ðŸ”—
    So what is an antiderivative of the integrand \(y = t\left(1 - t\right)^2\text{?}\)
    ðŸ”—
    ðŸ”—
    Solution.
    \begin{align*} \int_{-1}^1 t\left(1 - t\right)^2\, dt \amp= \int_{-1}^1 t\left(1 - 2t + t^2\right)\, dt \\ \amp= \int_{-1}^1 \left(t - 2t^2 + t^3\right)\, dt \\ \amp= \left[\frac{t^2}{2} - \frac{2t^3}{3} + \frac{t^4}{4}\right]_{-1}^1 \\ \amp= \left(\frac{1^2}{2} - \frac{2(1)^3}{3} + \frac{1^4}{4}\right) - \left(\frac{(-1)^2}{2} - \frac{2(-1)^3}{3} + \frac{(-1)^4}{4}\right) \\ \amp= \frac{1}{2} - \frac{2}{3} + \frac{1}{4} - \frac{1}{2} - \frac{2}{3} - \frac{1}{4} \\ \amp= -\frac{4}{3} \end{align*}
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Note 5.4.7. What if I forgot the \(\boldsymbol{dx}\text{...}\) Is it a big deal?

    Observe that the \(dx\) didnâ€™t contribute to the work at all... Is it okay to drop the \(dx\text{?}\)
    ðŸ”—
    The short answer is no... Dropping the \(dx\) will make the answer \(\pm\infty\) (ish...).
    ðŸ”—
    Remember the definite integral is defined using the limit of Riemann sum, where the \(dx\) is the infinitesimally small width of the rectangles. Without multiplying the \(dx\) with the function value (aka the height of the rectangle), we donâ€™t get the area of the rectangles out of it. We would be just adding up infinite number of function values together, resulting in \(\pm \infty\text{.}\)
    ðŸ”—
    Make sure you have the \(dx\) in all the definite integrals to make sure you are indeed finding the area under the curve, rather than simply adding all the functions values together.
    ðŸ”—
    ðŸ”—
    Now that we have an idea on how to evaluate definite integrals with the integrand being a single-piece functions. What about evaluating definite integrals with the integrand being a piecewise function? Letâ€™s try two of them!
    ðŸ”—

    Example 5.4.8.

    Evaluate the definite integral \(\displaystyle \int_{-2}^3 f(x)\, dx\text{,}\) where
    \begin{equation*} f(x) = \begin{cases} 12 - x^2 & \text{when } x \leq 2 \\ x^3 & \text{when } x > 2 \end{cases} \end{equation*}
    ðŸ”—
    Hint.
    Observe that we have a different function formulas before \(x\) hits \(2\) and after \(x\) hits \(2\text{.}\) So we donâ€™t know which formula to use to represent the function on the entire interval of \([-2,3]\text{.}\)
    ðŸ”—
    It would be so nice if we could split up the region at this transition point of \(x = 2\) so we can use the first formula on \([-2,2]\) and the second formula on \((2,3]\text{...}\) Is there a property of definite integral that will allow us to split up the definite integral?
    ðŸ”—
    ðŸ”—
    Solution.
    One of the properties of definite integral TheoremÂ 5.2.16 allows us to split up the definite integrals at \(x = 2\) as follows:
    \begin{equation*} \int_{-2}^3 f(x)\, dx = \int_{-2}^2 f(x)\, dx + \int_2^3 f(x)\, dx \end{equation*}
    Ha! We know that \(f(x) = 12-x^2\) on the interval \([-2,2]\) and \(f(x) = x^3\) on the interval \((2,3]\text{!}\) Then we can plug in the corresponding function formulas and let FTC to do the rest of the work!
    \begin{align*} \int_{-2}^3 f(x)\, dx \amp= \int_{-2}^2 f(x)\, dx + \int_2^3 f(x)\, dx \\ \amp= \int_{-2}^2 \left(12 - x^2\right) \, dx + \int_2^3 x^3\, dx \\ \amp= \left[12x - \frac{x^3}{3}\right]_{-2}^2 + \frac{x^4}{4}\bigg|_2^3 \\ \amp= \left(12(2) - \frac{2^3}{3}\right) - \left(12(-2) - \frac{(-2)^3}{3}\right) + \frac{3^4}{4} - \frac{2^4}{4} \\ \amp= \frac{64}{3} - \left(-\frac{64}{3}\right) + \frac{81}{4} - 4 \\ \amp= \frac{707}{12} \end{align*}
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Example 5.4.9.

    Evaluate the definite integral \(\displaystyle \int_0^\frac{3\pi}{2} \left|\sin(x)\right|\, dx \)
    ðŸ”—
    Hint.
    What makes this problem difficult is the absolute value... We donâ€™t have any integral formula to help us find the antiderivative of the absolute value function.
    ðŸ”—
    It would be so nice if there is a way to remove the absolute value...
    ðŸ”—
    There is actually a way to do so! Remember absolute value function is essentially a piecewise function as follows:
    \begin{equation*} |x| = \begin{cases} x & \text{when } x \geq 0 \\ -x & \text{when } x < 0 \end{cases} \end{equation*}
    In other words, the absolute value turns all the negative values to positive.
    ðŸ”—
    So do we know where, within \([0,\frac{3\pi}{2}]\text{,}\) the sine function takes on negative value?
    ðŸ”—
    ðŸ”—
    Solution.
    Recall that the value of the sine function is positive on \((0,\pi)\) and negative on \(\left(\pi,\frac{3\pi}{2}\right)\) (if this fact doesnâ€™t click instantly, graphing out the sine function on \(\left[0,\frac{3\pi}{2}\right]\) and observe the graph). Then we can remove the absolute value by rewriting the integrand as a piecewise function as follows:
    \begin{equation*} \left|\sin(x)\right| = \begin{cases} \sin(x) & \text{when } 0 \leq x \leq \pi \\ -\sin(x) & \text{when } \pi < x < \frac{3\pi}{2} \end{cases} \end{equation*}
    Then we can split up the integral at this transition point of \(x = \pi\) and plug in the corresponding function formulas. The FTC will do the rest!
    \begin{align*} \int_0^\frac{3\pi}{2} \left|\sin(x)\right|\, dx \amp= \int_0^\pi \left|\sin(x)\right|\, dx + \int_\pi^\frac{3\pi}{2} \left|\sin(x)\right|\, dx\\ \amp= \int_0^\pi \sin(x)\, dx + \int_\pi^\frac{3\pi}{2} -\sin(x)\, dx \\ \amp= -\cos(x)\bigg|_0^\pi + \cos(x)\bigg|_\pi^\frac{3\pi}{2} \\ \amp= -\cos(\pi) - \left(-\cos(0)\right) + \cos\left(\frac{3\pi}{2}\right) - \cos(\pi) \\ \amp= 1 + 1 + 0 + 1 \\ \amp= 3 \end{align*}
    ðŸ”—
    ðŸ”—
    ðŸ”—
    Again, make sure you are comfortable with evaluating definite integrals in this way. We will be developing more and more fancy methods to evaluate definite integrals using FTC. A good basic understanding and fluency in this way will make the later sections easier!
    ðŸ”—
    There is a saying "practice makes perfect". A good way of practicing the FTC is to go back to all the examples we did and see if you can solve them without looking at my solution. I will also include some practice problems at the end you can work on.
